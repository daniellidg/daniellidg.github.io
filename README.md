### 本地启动
```
bundle exec jekyll serve
```
然后在浏览器打开以下链接就可以访问了：

[http://localhost:4000/](http://localhost:4000/)



如果你用的是Linux系统或者Mac那么恭喜你，可以省去一些环境的配置。而如果你是Windows系统，下面这个链接教程还是建议你看一下。
